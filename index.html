MG-DOS

Functional OS in Python

About MG-DOS

MG-DOS is a functional operating system written in Python. It provides a simple command-line interface (CLI) with various commands to perform tasks like file operations, text manipulation, encryption, decryption, running scripts, and more.

Key Features

  • Simple and easy-to-use command-line interface
  • Support for various commands for file and text manipulation
  • Encryption and decryption of messages
  • Calculator functionality
  • Compare numbers
  • Listen to audio files
  • File system tree and directory browsing
  • Creating and deleting files and folders
  • Run script files
  • And more...

Usage Instructions

To use MG-DOS, simply type the commands in the terminal or console window. Below are some of the available commands and their usage:

help
Displays available commands and their usage. To see specific command usage, type help <command_name>.
write [filename]
Write to a text file with the given filename. The OS will prompt for content to write to the file.

Get Started

To start using MG-DOS, download the source code and run the Python script in your terminal or console.
